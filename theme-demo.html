Como Usar

1. Importar CSS

<!-- Ordem importante -->
<link rel="stylesheet" href="design-tokens.css">
<link rel="stylesheet" href="themes.css">
<link rel="stylesheet" href="index.css">

2. Importar Script

<script src="theme-manager.js"></script>

3. Aplicar Tema

// Aplicar tema por nicho
updateTheme('restaurant');

// Aplicar tema + customizações
updateTheme('barbershop', {
  '--btn-primary-bg': '#d4af37',
  '--accent-primary': '#c4952e'
});

// Obter tema atual
themeManager.getCurrentTheme();

// Exportar configuração (para salvar no banco)
const config = themeManager.exportThemeConfig();

4. Escutar Mudanças

document.addEventListener('themechange', (e) => {
  console.log('Novo tema:', e.detail.theme);
  console.log('Overrides:', e.detail.overrides);
});

Arquitetura em 2 Camadas

Layer 1: Base Tokens

Cores brutas da paleta (sem uso direto em componentes)

--color-neutral-100: #f5f5f5;
--color-neutral-500: #a0a0a0;
--color-brand-500: #E5B25D;
--color-success-500: #00B894;

Layer 2: Semantic Tokens

Tokens com significado (ÚNICA camada usada no CSS)

--bg-page: var(--color-neutral-900);
--accent-primary: var(--color-brand-500);
--btn-primary-bg: var(--color-brand-500);
--text-primary: var(--color-neutral-000);
